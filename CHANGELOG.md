# Changelog – Re-Elect Scott Brummer Campaign Website

## [1.0.0] – 2026-02-18

### Added
- `index.html` – Full homepage with hero, ticker, about strip, priorities grid, pull quote, donate CTA, contact strip, and footer
- `about.html` – Full about page with story section, photo frame, values band, issues list, leadership record grid, and CTA
- `endorsements.html` – Full endorsements page with featured cards (Snaza, Abbarno, Riley), WA Farm Bureau org endorsement, community voices grid, and support/endorse form
- `contact.html` – Full contact & donate page with quick contact cards, contact form, volunteer form, donate section with amount buttons, yard sign CTA
- `assets/images/` – Directory for candidate and campaign photos (placeholder)
- `assets/css/` – Reserved for extracted stylesheets
- `assets/js/` – Reserved for extracted scripts
- `robots.txt` – Search engine directives
- `sitemap.xml` – XML sitemap covering all 4 pages
- `README.md` – Project overview, structure, configuration checklist
- `CHANGELOG.md` – This file
- `DEPLOYMENT_GUIDE.md` – Step-by-step deployment instructions
- `FILE_MANIFEST.md` – Complete file listing with descriptions
- `.gitignore` – Standard web project ignore rules
- GitHub Pages enabled at `https://syncshepherd-main.github.io/Brummer/`
